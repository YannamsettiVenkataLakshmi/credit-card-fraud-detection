## Dataset

This project uses the Credit Card Fraud Detection dataset from Kaggle.

🔗 Dataset source:
https://www.kaggle.com/mlg-ulb/creditcardfraud

Due to licensing and size constraints, the dataset is not included in
this repository.

### How to use the data

1. Download 'creditcard.csv' from the Kaggle link above
2. Place the file in this folder:data/creditcard.csv
3. Run the notebook 'credit_card_fraud_detection.ipynb'
